# ThereWeather

![LOGO](https://user-images.githubusercontent.com/72483407/141734266-3097d834-b49e-4444-a2bd-6a1a293c2464.png)

- 사용자 간의 사진 및 정보 공유를 통해 부정확한 날씨 예보에 대비하는 애플리케이션입니다.
- 동네 사람들이 직접 작성한 제보를 통해 사진, 체감 온도, 바람 세기, 추천 옷차림을 확인할 수 있어요.
- 지도 페이지에서 위치별 게시물을 확인할 수 있어요.
- 기상청의 예보와 제보 데이터를 비교할 수 있어요.
- 직접 제보를 작성할 수 있어요.

## 미리 보기

### 메인
- 현재 위치 기준으로 유저가 제보한 날씨 현황, 기상청 날씨 예보, 적합한 옷차림, 게시물을 확인하세요.
<img width="80%" alt="mainpage" src="https://user-images.githubusercontent.com/61111156/148693109-4d4d1341-51d6-4007-82bf-493ce77197d1.png">

### 지도
- 지도를 드래그, 확대하거나 주소를 검색하여 위치별 게시물을 확인하세요.
- 지도의 마커는 게시물이 작성된 곳의 위치입니다.
- 우측의 박스에 게시물 목록이 표시됩니다. 클릭하여 게시물을 읽어보세요.
<img width="80%" alt="mappage" src="https://user-images.githubusercontent.com/61111156/148693112-bf2f354f-6550-4da5-b078-837f39e34ef7.png">

### 게시글 읽기
- 게시물 미리 보기 카드를 클릭하면 읽기 페이지로 연결됩니다.
- 큰 사이즈의 사진, 게시물의 내용을 확인하세요.
- 마음에 드는 글을 찾았다면 제목 오른쪽의 하트 버튼을 눌러 북마크하세요.
<img width="80%" alt="readng" src="https://user-images.githubusercontent.com/61111156/148693124-d101c8d0-d4f8-4751-8fb0-3e8991bdb978.png">

### 북마크 페이지
- 북마크한 게시물을 확인하세요.
- 카드를 클릭하면 읽기 페이지로 연결됩니다.
<img width="80%" alt="bookmark" src="https://user-images.githubusercontent.com/61111156/148693117-9c6e7196-cdf6-4477-8c5b-d754e471ad5a.png">
